이미지지정

background-image 속성으로 요소의 배경을 이미지로 지정

배경이미지의 크기가 요소 영역의 크기보다 작으면 반복적으로 이미지가 채워짐

 
 
 
 

이미지 크기 설정

background-size 속성으로 이미지의 크기 설정

단, 이미지의 크기를 지나치게 키우면 이미지가 매끄럽지 않게 출력됨

참고로 이미지의 계단현상을 줄이는 기능을 anti-aliasing이라 함

 
 
 

이미지 위치 지정

background-position 속성으로 영역내 이미지의 위치 지정

 
 

이미지 스크롤 고정

background-attatchment 속성으로 화면스크롤시 이미지를 고정시킬 수 있음

background 축약

설정순서는 color, image, repeat, attachment, position

 

sprite

스프라이트란 여러 개의 개별이미지를 하나의 이미지에 합쳐서 관리하는 이미지를 의미

웹 페이지에 이미지가 포함되는 경우 이것을 화면에 표시하기 위해 서버에 요청해서 해당 이미지를 다운로드 해야함

이미지가 많을수록 서버에 요청하는 횟수 증가 => 웹 페이지 로딩시간 증가

따라서, 스프라이트를 사용하면 서버에 이미지 요청횟수를 단 몇번으로 줄일수 있음

또한, 다량의 이미지파일 관리의 용이성도 있음
