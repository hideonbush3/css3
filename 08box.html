CSS 박스 모델

개요

모든 html요소는 박스로 구성되며 이를 박스모델이라 함

박스모델은 content, padding, border, margin으로

웹 디자인의 핵심은 컨텐츠를 담을 박스모델(블록, 인라인, ...)을 정의하고 CSS속성으로 스타일과 위치, 정렬을 수행하는 것

박스모델을 통해 텍스트, 이미지, 테이블등의 요소를 배치할 수 있고 이것을 통해 웹페이지의 레이아웃을 구성할 수 있음
