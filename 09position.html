CSS position 관련

개요

지금까지는 margin/padding 또는 인라인/블록 요소의 특성에 의해 부분적으로 요소의 위치를 지정할 수 있었음

반면, position이라는 속성을 이용하면 html 요소들의 위치를 좌표등으로 결정할 수 있음

즉, top/left/right/bottom 등의 속성으로 요소의 위치를 정할 수 있음

static position(정적 위치 설정)

기본적인 요소 배치순서에 따라 위->아래, 좌->우로 배치되며, 부모-자식 관계의 요소는 부모요소에 따라 배치됨 기준 없음 (배치 불가능 / 기본값)

box#1
box#3

absolute position(절대 위치 설정)

부모(조상) 요소를 기준으로 top/left/right/bottom만큼 요소를 이동시켜 배치

단, 부모요소를 기준으로 배치하려면 부모요소의 position을 relative로 설정해야함

box#4
box#5
box6
box#7

relative position(상대 위치 설정)

요소 자기 자신을 기준으로 top/left/right/bottom만큼 요소를 이동시켜 배치

box#8
box#9
box#11

fixed

뷰포트(웹페이지 가상영역)를 기준으로 top/left/right/bottom만큼 요소를 이동시켜 배치

스크롤해도 뷰포트를 기준점으로 붙어있다

position:fixed이며 viewport 기준으로 위치 조절할 수 있고 화면에 고정된다

sticky position

relative position과 fixed position을 적절히 조합해서 만든 position

평소에는 static 방식으로 요소를 처리하다가 해당 요소가 뷰포트를 지나칠 때 fixed처럼 화면에 고정시킴

z-index

먼저 만들어진 요소는 나중에 만들어진 요소 밑(뒤)에 배치됨(겹쳐서 보면 깔린다)

요소가 쌓이는 순서를 강제하기 위해서 z-index를 사용함

z-index값이 클수록 위에 위치함

z-index값은 음수도 지정 가능
